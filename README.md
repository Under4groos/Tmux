# Install 
```
sudo apt-get install tmux
```





# Split frame
Horizontal
```
tmux split-window -hf
```
Verical
```
tmux split-window -vf
```
KeyBinds
```
ctrl + b + % - to make a vertical split.
ctrl + b + " - to make a Horizontal split.
ctrl + b +   - left arrow to move to the left pane.
ctrl + b + " - to make a Horizontal split.
```
